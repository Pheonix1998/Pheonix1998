### 1. 🏃‍♂️ HealthTech User Engagement & Retention Analysis  

<a href="https://github.com/TATHAGATA23071998/PROJECTS/tree/main/STRAVA%20PROJECT%20FILES">
<img src="https://img.shields.io/badge/Project_Repository-181717?style=for-the-badge&logo=github&logoColor=white">
</a>

**Problem:** Declining engagement and mid-week user drop-offs were impacting retention and Customer Lifetime Value (CLTV).  

**Analysis:** Consolidated multiple datasets into a unified analytical layer and examined user behavior across activity, sleep, and engagement patterns.  

**Key Insights:**  
- Identified a retention spike on **May 12 (International Nurses Day)**, highlighting healthcare workers as a key segment  
- Linked **low sleep duration (~5.8 hrs)** with reduced mid-week engagement  
- Discovered **fixed engagement windows (12–14 hrs, 17–19 hrs)** driven by user routines  

**Recommendation:** Align engagement strategy with user behavior—optimize notification timing, introduce recovery-focused features, and target high-value user segments.  

<details><summary><b>🛠️ Technical Execution</b></summary>

**Tools:** Python | SQL Server | Tableau | ETL Design  

Developed a structured ETL pipeline integrating multiple datasets into a daily-level master table, with context-aware imputation to preserve data integrity.

### 2. 🚖 Ride-Hailing Operations: Revenue Leakage & Supply Bottlenecks  

<a href="https://public.tableau.com/app/profile/tathagata.chakraborty5102/viz/RIDESERVICEANALYSISDASHBOARD/Dashboard">
<img src="https://img.shields.io/badge/Project_Repository-181717?style=for-the-badge&logo=github&logoColor=white">
</a>

**Problem:** A high ride failure rate (~38%) was causing significant revenue leakage, requiring clarity on whether the issue was demand- or supply-driven.  

**Analysis:** Modeled and analyzed 100K+ ride records to evaluate cancellations, wait times, and payment behavior.  

**Key Insights:**  
- Bottleneck confirmed as **supply-side**, driven by driver cancellations  
- Identified **driver gaming behavior** impacting service reliability  
- Found **cash payments** correlated with higher cancellation risk  

**Recommendation:** Introduce operational controls (GPS-based reassignment), incentivize digital payments, and improve driver ecosystem support.  

<details><summary><b>🛠️ Technical Execution</b></summary>

**Tools:** SQL Server | Python | Tableau | Streamlit  

Built a star schema for efficient querying, handled unstructured data via junk dimensions, and optimized performance for BI dashboards.  

</details>
